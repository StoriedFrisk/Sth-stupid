无题

发表于2025-09-09|更新于2025-09-09
|阅读量:

再一次告别上海之后，我略有遗憾地在高铁卧铺上看着手机，向我的好友发送着颇有感情的话语。却收到了这些天来最让我感动的话。
他决定不远千里来到我家来找我。

同样听首歌吧！

8月21日

18:00

阔别多日的朋友，莉莉白，今天终于要到达我的城市——浦城了。
其实，先前两天他就说要来了，不过因为一些事情搁置了两天，虽然我颇有言辞，但是当他说要到的时候，还是喜笑颜开了。
在被放了两天鸽子后，我百无聊赖而又慵懒地，在房间中等待他到站的通知。
目光所及的手机亮了起来。
…
似乎他到了。
该出发了。

19:00

简单地介绍了房间位置，和他闲聊几句，准备吃饭。
他点了几杯饮品，令人感叹，真是轻车熟路啊。

1：00

早有了解到莉莉白是晚睡的人，今天就选择稍微熬下夜吧。
让他使用我的笔记本，毕竟没有电脑的日子令人头疼，不知道他是不是和我一样这么想的呢？
…
和莉莉白度过了一段时间。
…

8月22日

9：00

睁开眼，似乎有点劳累，不过应该起床了。
只是莉莉白一定还需要睡吧。
真是贫弱，如果是我，一定要努力适配主人的生活习性。
在那之前，先和母上准备下午餐吧。

顺带一提，Dr.Peper刚入口十分难喝，不过喝到最后似乎喜欢了上了它。

13：00

窗外阴沉了下来，下起了小雨，虽然幽静地令人愉悦，但是本打算此刻带莉莉白出去转转，想来只能作罢。
但也不失为一件好事。
先叫他起床吧。
…
由于起的晚，母上和父皇已经用餐完了，包括我。
陪着莉莉白吃饭，消耗了一段时间。

14：00

淅淅沥沥的雨在外面拍打着砖瓦。
看来下午不能出门了，略有遗憾，希望不要影响到今天的出行。
就来展示下ps3吧。
我们主要探索原主机版和Java版的不同。
玩了滑翔，pvp，以及掘一死战的小游戏。
还一起玩了pvz，僵尸真超模呀。
(其实是我被爆扣了)
和朋友一起玩主机游戏，真是曾经幻想却又无法实现的愿景。

18：00

莉莉白似乎要尝试打车，我自然是不信的，小小的浦城没有那么便利的交通，手机打车自然也是做不到的吧。
不过他依然还是尝试了，也罢，就让我在一旁看看他失望地喊出“omg“吧。
==顺带一提，”omg”是他的小口癖，大概像我的”tf”? #619AC3==
哼哼，城市里的人终归是要吃点苦头的。
…
然而我却迎来了当头棒喝，居然有人愿意接单。
在这样偏僻的城市，这样一单仅收6块钱，居然也有司机愿意么？
莉莉白真用他的经验让我意识到了其实科技是能走进山村的。

19：00

来到了永辉商城。
我向来不习惯这样商业气息浓重的地方，但是对于他来说就不一样了，如鱼的水的他，终于在商城这样与城市相似的一隅，如含羞草一般舒展开来了。
一楼有他熟悉的店铺——霸王茶姬
我在店里歇息，他熟练地下单，并取得饮品，如果是我的话，即便有尝试的念头，也会很快因为不熟悉便打消吧。
稍微看一下宣传单，他就从柜台回来了。
我先前从来没有喝过霸王茶姬，这次终于试了试。
嗯，想不到奶茶居然真的会有奶茶香，也许我下次应该试试？
说罢，先找找舞萌吧，据我的朋友说，此处应该是有舞萌的，正好见识一下传说中的舞萌是什么样的。
然而出乎意料的是，电玩厅并没有舞萌的身影，也许是我记错了。
令人神伤。
不仅如此，甚至永辉超市——一大探索永辉的乐趣——都暂时关门了，似乎九月十二日才会重新开业。
…决定出去逛逛。
在出永辉不超过五分钟脚程的地方，发现了一家糖巢。
似乎是一家零食店，从来没有去过。
展现着Cityboy实力的莉莉白，邀请我走了进去。
其中的食物，以惊人的，舒心的价格吸引着我，说清楚点的话，就是便宜。
于是开始了一场锐评零食的大会——我们一边锐评着食物的性价比，一边展露着对某些零食的喜爱。
尽情地在之中消耗了一段时间。

从糖巢出来之后，顺着二中的路决定回家。
“这路上有很多奶茶店哦，有七家那么多！”
“omg”，又是他的经典台词。
“不过我不怎么喝奶茶啦，似乎像是女生喝的东西”
然后有点贵。
“那一定有蜜雪冰城吧”
“去试试冰淇淋吧！”

在前往的途中…
”那就是我的学校哦“
“欸，你以前也会藏手机吗？”
“…我把它藏到了空调后面，根本没有人会发现！”
“吼吼，我可是放到了窗帘的夹层里面！”
没想到我居然会和他聊以前藏手机的经历，看来各地的高中都是一样的呀。
“他们也是懂的，特意选了SE这种机子。”
嗯，大家都是一样的。

前往了蜜雪冰城，
在我还在思绪卡壳，断断续续的思考该先扫码还是询问时，莉莉白已经完成了购买。
展示了购买两块钱的冰淇淋的可能性后，二人接着探索着商业城。
大概是因为第一次吃，吃相颇为难看，我将手沾满了冰淇淋。
果然，莉莉白吃的完美又干净呀。

两人谈天说地地走到了商业城仓储式超市下，决定前去看看。
“我喜欢走进超市逛，不过我总是会买一个东西，算是一个习惯咯”
这家所谓仓储式超市，说到底不过是仓库，但是顺带售卖商品罢了。
本想着给莉莉白推荐一袋桂花回去。但零售的桂花都相当昂贵。
暗自扶额后走出了超市。

夜晚的商业城，相较于几年前，其实并没有那么热闹，毕竟曾经的小吃街已经不在了。
此处充斥的不再是人群，而是满地的遗憾。
我和莉莉白穿越了道路，他随我在我身旁，而我以二人间的谈资消斥着满城的遗憾。

来到了上水南的小路，矗立在这的是一座对我来说熟悉万分的教堂。
一种诡异的神圣感和探险的感觉萦绕着我二人。
难道这里是寂静岭？！

从教堂右转，莫名地放松了一口气。
“这里的天，真的是全黑的呀，如果是我家的话，晚上的天空是暗黄色，发亮的呢。“
我不禁想起福州，上海等城市的地方，似乎确实如此。
便和他一起抬起了头，看向了天空——
布满着闪烁的星星。
我头向后一仰，这是我从来没见过的光景。
星星点缀着纯黑的幕布，如同草地上的朵朵红彦。
又好像幻想一般的感觉，虚幻但却真实。
好像密封一般。

我和他在原地沉浸了许久，期间不直地发出惊叹。
因为不管是高动态范围拍出的照片，还是极具小城的情景也好，都敌不过真正的天空的颜色。
尝试着用手机留下影像，也不过是如下的悲剧般的照片。

为了能在地上继续这不长的，的平凡琐碎的闲谈，两人离开了教堂。

8月23日

1：00

莉莉白精力真的很旺盛呢…
和他度过了很长一段时间…

下午1点

在我的劝说之下，今天决定去仙楼山。
不知莉莉白是不是不喜欢这样自然气息太浓的地方呢？
还是，这样炎热的下午，让他难以挪动了双脚呢？
同样试图打车前往了仙楼山，这一次终于学会在路口打车了。
稍稍的踏上了上山的楼梯。

从健身器材到高铁建设。天南地北不过我们嘴中的小小谈资，不一会我们便登上了山顶。
山顶是革命英烈纪念碑，虽然我们没有合照，但是天子fumo留下了照片作为纪念。

决定下山。
下山过程中，我们尝试了走不同的道路。

即便不认途中的路，大概也能在山脚寻到目标，所以我并不担心。
偶然走到了浮桥路口，回到2016年，我和家人们在这走过。
夏日的下午依然很炎热，于是有几位勇士在水中搏斗浪花。
莉莉白在浮桥之上轻快地走了过去。
波光粼粼的水映射这夏日最后几天的光芒无限。
我拿着的天子fumo吸引着一旁的人，他热情的询问我为什么会来这，以及fumo的名字。
如今真少见热情的人。
告别了他之后，我向着莉莉白，朝桥对岸走去。
浮桥
桥对岸是我从来没去过的地方，相比水南，这里看着更像乡村了。
路旁有传统的小卖部，而非便利店。泥胚房和新式农村居民楼交错，它们的共同点是都有鸡窝鸭圈。水泥路上还常有稻谷，路旁的田居然是最不像乡村的东西。要不是远处的高楼，我真以为我们穿越了。

漫步在乡村不失为闲情逸致。

但即使是八月末，炎热的夏天还是能燃烧我们散步的意志。
在订的车来之前，好好感受乡村的风吧。

17：00

乘坐出租车来到了上次访问永辉偶遇的糖巢。
本因为看见窗户已经关闭，以为关门的糖巢Turns out to be只是在关闭窗户避免阳光直射饮料。真是令人欣喜。
稍稍购买了一些补给，准备考虑回家吃饭。
在刚出门的时候，母上打来了电话，询问何时归家。
…
思考片刻，我决定让今天变得更长一些，就这样前去素菜馆，带他吃一下自助吧。
我们沿途高谈阔论，不禁让我想起中学那段和好友相同走过的日子。
如今能再有同龄人莉莉白在我身边闲聊，就像冬日的一杯热咖啡。
带给我的温暖是一样的。

18：00

到达了素菜馆，和莉莉白评价了一下菜品，虽然我已经来很多次了，可是与同龄人一起吃饭的经历是极少的。
我思考这是一种什么样的感觉，是久久的，没有品尝过的，他人能够理解你的感觉，它与亲友间那种感觉截然不同。尤其是，当想到他跨过千山万水来到本地时，意识到这样的经历难得且珍贵，我便更加感到温暖了。
在我的建议之下，莉莉白与我一起前往小米之家看看能不能修理我的手环。
即使是去小米之家这样一件小事，与同伴一起去也是不同的吧，当你知道他和你同样了解这个地方，甚至有许多地方可以请教。那是一种真正的他山遇知音的写照。
在知道修理手环不值过购买新的之后，我俩遗憾地走出了店面。
沿途介绍了我的小学，走过了小巷，看见了很老的木制建筑。
跨过了小桥，走过了曾经我放学的小路。
曾经我与我的朋友们在无数个放学路过了这里，像今天这样在黄昏与夜晚的交界，还是第一次吧。
但是同样的是，我在与朋友一起走，我最珍视的朋友之一。
就这样回家吧…
按照着约定，莉莉白将和我尝试Wii！
作为一台家庭娱乐主机，PS3在连接家人这方面做的完全不如Wii。
在一个人的旅途中，各种运动游戏，马里奥，都被我玩了个遍，说是完美通关也不为过。
然而，和朋友一起度过的时光，让Wii不再是主体，而是一位朋友间的管家。
仿佛切身于运动场上的时间，让我这样的御宅，也能体验到摸草的感觉。
休息的时候，回到了房间里。
在闲暇的夜晚，打开了Minecraft基岩版，尝试邀请了莉莉白一起玩。
不得不提微软抽象的uwp软件，基岩版无论如何也无法更新。
但是莉莉白的恒心比我强多了，就让这样子等着他找解决办法，而我在一旁游玩游戏的，伴着有着没得吐槽，难得，难忘且惬意。
闲时，他和我点了一份汉堡吃，让这段时间变得更加珍贵且美丽。

8月24日

1：00

时间似乎不多了，大概，明天就要和莉莉白离开了吗。
不知道他现在在想什么呢…

10：00

在早上我来到电脑前，偶尔瞥见macOS的通知。
-莉莉白的生日 八月二十五日
就像其他特别的朋友一样，我会为他们写一份祝贺邮件。
…….就设在…第二天，零点吧!
如果一切顺利，莉莉白这个时候应该到家没多久吧。
突然手机响了起来。
是莉莉白找我了。
今天这么早醒吗？

似乎，学校告诉莉莉白明天需要他回去一趟。
如果今天走的，还需要回学校的话，还不如第二天再走。
但如果26号再走，平静的陪伴爸妈的日子似乎也少了吧。
和莉莉白一样，母上也是花了很大的力气，才能有一个月时间陪我。
两难的境界困着我和莉莉白，我与他在床上看着天花板，看着天花板的时候，最容易逃离现实的困境。

16：00

展示了黄昏公主。
堪称最写实的塞尔达作品了，但是对莉莉白来说果然还是太大果粒了。
老游戏的缓慢节奏，很适合这样平静的日子，在宁静的日子和朋友慢慢看着云朵缓缓走过。
今天还剩多少呢…

夜晚的二人依然有说有笑，全然没有第二天就要离开的样。
这就是我需要的…和朋友间惬意的时光，不再孤独的感觉。
只是时间流逝…今天，大概不剩下多少了吧…

8月25日

2：00

休息的时间果然还是要到了吗。
但是在太阳升起之前，二人都不必面对离别的伤。
和他谈论着所喜欢的，所厌恶的，惊人的一致…
逐渐忘却了在哪停下，缓缓变亮的天空在我眼里逐渐暗淡下来…

6：00

提醒着他起床，似乎比前两天更加困难呢…
对我，也并不容易。

过去在远离 是否意味着未来正在靠近？
如今只能旁观着这段短暂的旅途的我，才能够觉察着这变化。
广袤的世界，渺小而又彼此独特的个体。
这种偶然的时间重合 已经形成了震惊宇宙的奇迹。
然而未来变为现在，现在已然过去。
守望着的那一到站的时候终于会到来，是朋友下站的时间。
站在站台挥手并非是我唯一能做的。
心怀理想并努力为下一次见面而奔跑。
用自己的双眼再去见证下一次的会晤吧。
…
不知司机什么时候到呢？
啊啊，是现在吗？
…
似乎他到了。
该出发了。

Forward thinking.
